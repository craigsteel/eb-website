Fusiondb Nightly Builds

These distributions are built nightly from the development head of the FusionDB Server source code. New builds will only appear when the source base changes, this means that there may not be a build for each day if there were no changes committed to the project on that day. The files are signed by Evolved Binary.

FusionDB is not Open Source. These FusionDB builds are made available under Evolved Binary's FusionDB Alpha License. These builds are suitable for non-commercial non-production purposes only. All builds require Java 8 or newer with JavaFX, and include eXist-db compatibitity modules.

eXist-db Nightly Builds

These distributions are built nightly from the develop branch of the the eXist-db project's GitHub: https://github.com/exist-db/exist. New builds will only appear when the source base actually changes, this means that there may not be a build for each day if there were no changes committed to the project on that day. The JAR files are signed by Evolved Binary and are provided to the community for their convenience. All builds require Java 8 and include all eXist-db extensions.
These eXist-db builds are made available under eXist-db's LGPL 2.1 license. All builds require Java 8 or newer, and include all eXist-db extension modules.
